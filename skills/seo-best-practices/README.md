# SEO Best Practices

Comprehensive SEO patterns for web applications built with React and Laravel. Supports both coding reference and SEO audit mode with PASS/FAIL output.

**Version:** 1.2.0

## Overview

- SEO audit with PASS/FAIL/N/A checklist output
- Core Web Vitals optimization (LCP, INP, CLS)
- Technical SEO fundamentals (meta tags, canonical URLs, sitemaps, robots.txt)
- Structured data with JSON-LD / Schema.org
- Social sharing with Open Graph and Twitter Cards
- React/SPA SEO (rendering strategies, dynamic meta tags)
- Performance optimization (images, fonts, lazy loading)
- Mobile-first indexing compliance
- 31 rules across 8 categories

## Categories

### 1. Core Web Vitals (CRITICAL)
LCP, INP, and CLS thresholds and optimization strategies.

### 2. Technical SEO (CRITICAL)
Meta tags, canonical URLs, robots.txt, XML sitemaps, and URL structure.

### 3. On-Page SEO (HIGH)
Heading hierarchy, semantic HTML, internal linking, and image alt text.

### 4. Structured Data (HIGH)
JSON-LD markup for Article, Product, BreadcrumbList, and other Schema.org types.

### 5. Performance SEO (HIGH)
WebP/AVIF image formats, lazy loading, font loading, and resource hints.

### 6. Social Sharing (HIGH)
Open Graph and Twitter Card meta tags for rich social previews.

### 7. React/SPA SEO (HIGH)
Rendering strategies, dynamic meta management, and crawlable routing.

### 8. Mobile-First (MEDIUM)
Viewport configuration, content parity, and mobile UX requirements.
